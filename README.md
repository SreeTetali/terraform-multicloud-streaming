# 🌍 Terraform Multi-Cloud Streaming Infrastructure

A short, educational example showing how to deploy streaming services across **AWS (Kinesis)**, **Azure (Event Hubs)**, and **GCP (Pub/Sub)** using Terraform modules.

This repo is designed for learning and as a portfolio piece. It contains:
- simple provider-specific modules under `modules/`
- runnable `examples/` for each cloud
- a clear README and MIT license
